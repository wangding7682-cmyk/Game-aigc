# 狙击识别伪装外星人·首批美术资产包

本批资产位于 `sniper-art-first-batch`，用于 Godot 狙击题材项目“狙击识别伪装外星人”的第一轮视觉落地。整体风格已经吸收了参考中的“软 3D 卡通、软塑料/软黏土质感、柔和高光、轻科幻”语言，同时把主题切换到“都市夜战 + 狙击 HUD + 目标识别”。

## 这一批资产包含什么

### 1. 武器商店卡图
- `assets/weapons/weapon-default-sniper.svg`
- `assets/weapons/weapon-precision-sniper.svg`
- `assets/weapons/weapon-auto-sniper.svg`
- `assets/weapons/weapon-plasma-sniper.svg`

用途：
- 可直接用于商店、武器库、升级界面、结算解锁弹窗。
- 也可以作为 Godot UI 中 `TextureRect` 的占位图或正式矢量源素材。

对齐配置：
- 标准狙击枪：金属灰，均衡基础款。
- 精准狙击枪：战术蓝，轮廓更细长，强调精确与稳定。
- 连发狙击枪：烈焰红，体块更厚重，强调高压连续输出。
- 电磁狙击枪：等离子绿，加入能量腔体和发光结构，强调未来科技感。

### 2. 目标识别角色卡
- `assets/characters/alien-target-card.svg`
- `assets/characters/civilian-decoy-card.svg`

用途：
- 可直接用于教程、图鉴、识别提示面板、任务说明、关卡简报。
- 也适合作为角色方向概念卡，供后续 3D 角色或 Spine/逐帧表现继续细化。

对齐逻辑：
- 外星人卡保留泪滴形大头、黑色大眼，并强化“红眼脉冲、肩线异常、可疑弱点提示”。
- 平民卡采用都市路人轮廓，冷蓝灰配色，肩线自然、姿态放松，和脚本中的识别特征保持一致。

## 简洁接入建议

### Godot UI 接入
1. 先把 SVG 直接导入工程资源目录。
2. 武器图与角色卡优先挂到 `TextureRect` 或卡片容器中。
3. 图标优先接入 HUD、商店按钮、道具栏。
4. 统一在暗色背景上使用，保留红色准星高亮与冷蓝辅助光。

### 使用建议
- 武器图推荐显示区域偏横向，保留四周留白。
- 图标建议放在深蓝黑面板上，搭配轻玻璃背景与发光描边。
- 角色卡建议配合“可疑点说明”文字使用，强化识别玩法记忆点。

### 简单测试建议
建议在以下真实业务场景先做一轮快速验证：
1. 武器库页面：检查 4 把枪在缩略尺寸下是否一眼可区分。
2. 战斗 HUD：检查 6 个图标在小尺寸下是否仍有可读性。
3. 新手教程：把外星人与平民卡并排展示，确认玩家能快速理解“红眼脉冲 / 异常肩线 / 自然肩线”的差异。
