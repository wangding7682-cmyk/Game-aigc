# 爆炸冲击三档与墙角崩边说明

## 本次新增

这次把障碍物爆炸响应从“单一冲击”升级成了两部分：

1. `wall_corner` 现在支持 **崩边 + 小块弹飞**
2. 爆炸冲击支持三档：
   - `light`
   - `medium`
   - `heavy`

---

## 当前三档定义

### `light`

适合：

1. 普通爆炸边缘扫到
2. 轻型武器附带冲击
3. 小范围技能外缘

表现：

1. 轻微位移 / 回弹
2. 少量飞散件
3. 更偏提示性

### `medium`

适合：

1. 当前默认阻挡命中升级版
2. 一般爆炸武器
3. 常规技能主命中区

表现：

1. 清晰的受力偏移
2. 明显的局部弹开件
3. 可推动路灯和广告牌进入更强响应

### `heavy`

适合：

1. 强爆炸武器
2. 技能核心爆心
3. 后续 Boss 或高阶破坏能力

表现：

1. 更强位移和抬升
2. 更多飞散件
3. `wall_corner` 会明显进入崩边态

---

## `wall_corner` 当前落地方式

当前不是整堵墙乱飞，而是：

1. 主体轻微受力偏移
2. 角部和边缘会飞出小块碎件
3. 重爆时会把破损状态抬高

这更适合当前项目，因为：

1. 仍然保留“这是稳定重遮挡”的认知
2. 玩家能明显看到“墙角被炸崩了一块”
3. 不会把教学关空间直接打散

## 代码接入点

### 掩体脚本

主要能力在：

- `pve_cover_obstacle_3d.gd`

关键入口：

- `apply_impact_feedback(hit_point, hit_normal, effect_type, blast_tier)`
- `apply_explosion_impulse(...)`

### 当前默认链路

在：

- `battle_mode_pve_3d.gd`

当前阻挡命中默认传的是：

- `blast_tier = "medium"`

也就是：

当前正式战斗里，如果没做额外武器区分，默认按 **中爆** 表现。

---

## 后续给武器 / 技能接法

后面如果武器和技能要接，只需要在命中掩体时决定两个字段：

1. `effect_type`
2. `blast_tier`

例如：

### 普通步枪

- `effect_type = "blocked"`
- `blast_tier = "light"`

### 榴弹或爆裂弹

- `effect_type = "metal"` 或 `concrete`
- `blast_tier = "medium"`

### 重型爆炸技能

- `effect_type = "concrete"` / `metal`
- `blast_tier = "heavy"`

---

## 当前建议

短期内建议这样配：

1. 普通子弹：`light`
2. 当前已有阻挡命中默认：`medium`
3. 明确爆炸技能 / 爆炸武器：`heavy`

这样能保证：

1. 普通射击不会过度炸场景
2. 常规爆炸有明显爽感
3. 重爆能力和普通命中有足够层级差
