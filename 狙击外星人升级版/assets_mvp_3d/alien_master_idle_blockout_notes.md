# alien-disguised-idle 标准 blockout 说明

## 本次产物
- `alien_master_idle_blockout.blend`
- `characters/alien_master_idle_blockout.glb`
- `alien_master_idle_blockout_render.png`

## 当前阶段
- 已从纯灰模阶段进入“带颜色的美术色块阶段”。
- 当前目标不是最终高精贴图，而是先把`头罩`、`面罩`、`衣服`、`手套`、`鞋`的颜色分层和材质关系立住。
- 颜色依据主要来自 `alien-disguised-idle.svg`：
- 头罩：灰蓝
- 外套：深灰蓝
- 主眼：粉红发光
- 鞋与手套：更深的近黑色
- Blender 内查看时，优先使用 `材质预览`，不要用纯 `实体显示` 来判断颜色和饰品层级。
- 如果在材质预览里仍然读不出饰品，就需要改成更强的`分件式外层`，不能只靠颜色暗示。

## 显性饰品化重构
- `头罩`：增加了更明确的侧边和下缘外层，不再只是一整颗头的外扩。
- `外套`：补强了前片、侧片和后侧外层，避免只像身体体块外鼓。
- `手套`：放大手套主体，并增加手腕口，让它更像单独套上的件。
- `鞋`：放大鞋体，并加鞋底和鞋头分件，让它和腿彻底分开。

## 新增参考方向
- 这轮额外吸收了你给的参考图里“第 3 排第 3 个”的方向。
- 这次吸收的不是那套具体服装本身，而是它的角色组织方式：
- 更大的头身比
- 更短更硬的四肢节段
- 脸部露出比例更集中
- 服装先组织身体，再决定表面细节

## 本轮服装逻辑
- `肩袖`：不再是独立肩垫，而是让肩部直接过渡进袖子，决定上身宽度和角色气质。
- `腰胯收束`：在腰线上做一次收紧，在胯部重新放开，让服装先形成 S 型节奏，再承接腿部。
- `短外套下摆`：不再用散开的前片单独表达，而是加统一下摆收口，中间再留分腿开口。
- `颜色显性化`：通过深色外套、浅色头罩/身体、粉色主眼，把衣服从身体层里明确剥离出来。

## 这次锁定的方向
- 不再直接改工程里正在跑的外星人资源。
- 先单独做一只 `alien-disguised-idle` 标准 blockout。
- 目标是先把“头罩 + 外套 + 分腿下摆 + 可读手脚”这 4 个大轮廓锁住，再决定是否回写工程资产。

## 第三轮后已经对上的点
- 头部是独立头罩阅读，不再是单纯柱状身体上接一个球。
- 身体改成了“外套包裹躯干”的阅读方式，和之前直接堆身体体块相比更接近伪装态。
- 下半身已经做出分腿，而不是整根立柱。
- 手、脚、肩部都被单独放大，便于在游戏中远景识别。
- 右肩体量略大于左肩，开始接近原稿里的不对称关系。
- 面部保留“大主眼 + 弱化副眼”的识别重点。
- 面罩从“圆球贴脸”继续往“罩壳开口”方向推进，正面识别更明确。
- 外套前片已经拆开，开始形成“外衣包裹、双腿从中间露出”的阅读关系。
- 胸前补了内层块和中缝，至少有了内外层关系，不再只是平面占位。
- 头罩增加了前缘边界，开始有“外壳罩住面罩开口”的层次。
- 面罩厚度继续压薄，前脸关系比第二轮更接近原稿。
- 手臂、小腿、脚口补了节段感，四肢不再完全像连续软管。
- 开始往“第三排第 3 个”那种大头小身、制服式包裹关系靠。
- 上半身从长外套感，收成了更像短外套/制服前片组织出来的体块。
- 身体裸露感被削弱，角色阅读更像“穿着服装的角色”，不是“身体外面罩一层东西”。
- 肩袖已经开始形成连续过渡，上身终于不再是“肩部一块、手臂一根”的拼接阅读。
- 腰线和胯部体量被拆开了，服装不再从胸口一路顺着落到腿根。
- 下摆已经有统一收口，中间保留分腿开口，整体更像一件短外套而不是几片零散衣摆。
- 已开始套入美术色块，角色不再只是无差别灰块。
- 头罩、面罩、服装、鞋手套已经进入分材质状态，后续可以在这个基础上继续细化。

## 第三轮后仍然没完全对上的点
- 头罩壳体已经有边界，但侧边还偏鼓，离原稿那种更利落的罩壳轮廓还有差距。
- 面罩虽然压薄了，但当前仍稍微偏大，和原稿相比还有一点“前伸”感。
- 外套下摆的前后包裹关系更清楚了，但整体仍偏 blockout，不是最终定形。
- 四肢已经有节段，但玩偶感还主要来自比例和附加体块，离真正精确的角色雕形还差一轮。
- 和“第 3 排第 3 个”相比，现在还缺更明确的腰胯收束，以及更利落的肩到袖子过渡。
- 服装已经有逻辑，但还没有形成更清楚的上衣/下装分界，当前仍偏 blockout。
- 肩袖虽然连起来了，但袖根和手臂之间还偏圆，离真正的服装裁片感还有距离。
- 腰胯收束已经成立，但侧面的服装外轮廓还不够利落，胯部仍略鼓。
- 短外套下摆已经统一，但还可以继续压平，让收口更干净。
- 当前还是“色块版美术皮相”，还没有做到贴图、缝线、磨损、布料细节这些最终表面层。

## 如果继续下一轮，建议只改 3 件事
1. 再收头罩两侧鼓包，让头罩更像完整罩壳，而不是球体改形。
2. 继续缩短面罩前伸量，并让右主眼位置更贴近原稿。
3. 在已成立的服装逻辑上继续定形：压袖根、收胯侧、压平下摆收口。

## 简单测试环节
1. 在 Blender 打开 `alien_master_idle_blockout.blend`，只看正视图剪影：头罩、肩宽、外套下摆、脚部是否都能一眼读出来。
2. 在引擎里只加载 `characters/alien_master_idle_blockout.glb` 做单体预览，不替换现网角色，确认远景下轮廓仍可辨认。
3. 把 `alien_master_idle_blockout_render.png` 和 `alien-disguised-idle.svg` 并排看，只判断大轮廓，不判断材质细节。
4. 额外看 3 个服装点：肩袖是否连成一体、腰线是否收出来、下摆是否是统一收口而不是散片。
5. 再看 1 个颜色点：头罩、衣服、手套/鞋、主眼是否已经能直接区分成 4 个层级。
