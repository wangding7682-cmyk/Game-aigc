# 《当前美术素材规范盘点》

> 版本：2026-06-30 校准版  
> 文档目标：只保留和当前 `Godot 3D MVP` 工程、最新角色方案、现行工程化规则能够互相对上的判断。  

---

## 盘点目的

这份文档现在只回答两个问题：

1. 现有素材是否已经支撑当前 MVP 可运行闭环
2. 哪些地方已经从“资源缺口”转成了“正式量产与规则收口问题”

不再沿用下面这些旧判断：

- “3D 还没真正接入主工程”
- “枪口火焰和反馈序列帧还没有消费”
- “武器仍停留在只有 2D 入口、尚未接入 3D”

这些判断在当前工程里已经不成立。

---

## 盘点依据

本次盘点对照以下来源：

1. `3d_asset_engineering/` 下的角色、掩体、反馈、贴花、武器、映射与校验规则
2. `已导入引擎的3D资源总清单.md`
3. `狙击外星人升级版` 当前 3D 工程脚本与可运行场景
4. `women_alien_minimal_feature_spec.md`
5. 当前 PVE 关卡配置、玩家手感分层说明与烟雾验证入口

---

## 总结论

### 结论一

如果只按当前 MVP 真实范围来判断，现有素材已经能够支撑：

1. 3D 搜索与瞄准
2. 目标识别与误判惩罚
3. 扫描、挡弹、命中、误判回放等反馈
4. 3 关配置驱动的 PVE 主循环

也就是说，当前问题已经不是“素材够不够跑起来”，而是“哪些占位资产要升级成正式资产”。

### 结论二

如果按规范覆盖度来判断，当前工程化规则已经形成完整主干，至少包括：

1. 角色占位规则
2. 掩体占位规则
3. 反馈规则
4. 贴花规则
5. 武器占位规则
6. 资产命名与接入规则
7. 主项目映射规则

因此这份盘点不再把“缺模块”作为主要问题，而是把注意力转到 `规则落地精度` 和 `正式资产替换顺序`。

接下来更值得做的不是继续补“占位有没有”，而是按下面顺序推进：

1. 把现有占位资产和真实挂点规则一一对齐
2. 先替换最影响战斗阅读的正式资产
3. 用同一套规范校对 PVE 主链路与预留 PVP 资产链路
4. 把资源状态从“已导入”进一步推进到“已正式替换”
