## 本次移除的旧口径

本轮校准明确移除了这些已经过时的判断：

1. 当前项目仍以 `Canvas/Phaser/H5` 为实现基线
2. 当前主设计已经包含 `Boss / Battle Pass / 段位 / 赛季 / 大商业化扩展`
3. 当前工程仍停留在“只有 2D 代理，3D 尚未真正落地”
4. 枪口火焰和反馈序列帧仍未进入消费链路
5. 武器仍然只有 2D 展示入口，尚未进入 3D 工程链路

## 当前最适合的使用方式

后续查文档时，建议按这个顺序：

1. 先看 `GDD_狙击大师_完整设计方案.md`
2. 再看 `Godot_MVP脚手架说明.md`
3. 再看 `当前美术素材规范盘点.md`
4. 最后按专题去看角色、资源、配置与工程化规则细文档

这样能先确定“项目现在是什么”，再去看“具体模块怎么做”。
